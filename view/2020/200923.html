JavaScript中的计算机科学：Quicksort

wangshanjian 2020-09-23 javaScript

正式的计算机科学程序也倾向于覆盖quicksort，这是因为它的时间复杂度（n log n）具有出色的平均复杂度，并且相对于其他效率较低的排序算法（例如针对大型数据集的冒泡排序和插入排序）而言，其相对性能有所提高。与其他排序算法不同，快速排序有许多不同的实现方式，它们会导致不同的性能特征以及排序是否稳定（等价项目按照其自然发生的顺序保留）。

Quicksort是合并排序样式的分而治之算法。基本思想是在数组中找到一个“枢轴”项以将所有其他项与之进行比较，然后移动项以使枢轴之前的所有项均小于枢轴值，枢轴之后的所有项均大于枢轴值。枢轴值。之后，对枢轴之前和之后的项目递归执行相同的操作。实现快速排序的算法有很多，本文仅探讨其中一种。

  1. 在数组中找到一个“透视”项。此项是单轮比较的基础。
  2. 在数组的第一项处开始一个指针（左指针）。
  3. 在数组的最后一项开始一个指针（右指针）。
  4. 当数组中左指针的值小于枢轴值时，将左指针向右移动（加1）。继续直到左指针处的值大于或等于枢轴值。
  5. 当数组中右指针的值大于轴值时，将右指针向左移动（减1）。继续直到右指针处的值小于或等于枢轴值。
  6. 如果左指针小于或等于右指针，则交换数组中这些位置的值。
  7. 将左指针向右移动一，将右指针向左移动一。
  8. 如果左指针和右指针不一致，请转到步骤1。

与许多算法一样，通过查看示例更容易理解分区。假设您具有以下数组：

var items = [4, 2, 6, 5, 3, 9];
1

有许多方法可以计算枢轴值。一些算法选择第一项作为枢轴。这不是最佳选择，因为它会在已排序的数组上提供最坏情况的性能。最好在数组的中间选择一个枢轴，因此将5设为枢轴值（数组的长度除以2）。接下来，在位置5的右指针（数组中的最后一项）的位置0处启动左指针。由于4小于5，将左指针移至位置1。由于2小于5，将左指针移至位置2。现在6不小于5，因此左指针停止移动，而右指针值为与枢纽相比。由于9大于5，因此右指针移动到位置3。值3不大于5，因此右指针停止。

接下来，左指针增加一，右指针减少一。这将导致两个指针都位于枢轴值（5）处。这表明操作已完成。现在，数组中位于枢轴左侧的所有项目都小于枢轴，而位于枢轴右侧的所有项目都大于枢轴。请记住，这并不意味着立即对数组进行排序，只是意味着数组有两个部分：所有值均小于枢轴的部分和所有值均大于枢轴的部分。

分区功能的实现依赖于swap()功能，因此这是代码：

function swap(items, firstIndex, secondIndex){
    var temp = items[firstIndex];
    items[firstIndex] = items[secondIndex];
    items[secondIndex] = temp;
}

分区函数本身非常简单，并且几乎完全遵循算法：

function partition(items, left, right) {

    var pivot   = items[Math.floor((right + left) / 2)],
        i       = left,
        j       = right;


    while (i <= j) {

        while (items[i] < pivot) {
            i++;
        }

        while (items[j] > pivot) {
            j--;
        }

        if (i <= j) {
            swap(items, i, j);
            i++;
            j--;
        }
    }

    return i;
}

此函数接受三个参数：items，这是要排序的值的数组left，，这是开始左指针right的索引，以及，这是开始右指针的索引。枢轴值是通过将left和right值相加然后除以2 来确定的。由于该值可能是浮点数，因此有必要进行一些舍入。在这种情况下，我选择使用下限功能，但是您也可以使用逻辑稍有不同的上限功能或舍入功能。该i变量是左指针和j变量是正确的指针。

整个算法只是一个循环。外循环确定何时处理数组范围内的所有项目。两个内部循环控制左指针和右指针的移动。当两个内部循环都完成时，将比较指针以确定是否有必要进行交换。交换之后，两个指针都移动了，以便外循环在正确的位置继续。该函数返回左指针的值，因为它用于确定下一次从何处开始分区。请记住，分区是在适当的位置进行的，没有创建任何其他数组。

快速排序算法基本上是通过对整个数组进行分区，然后以递归方式对数组的左右部分进行分区，直到对整个数组进行排序为止。数组的左右部分由每次分区操作后返回的索引确定。该索引实际上成为数组左右部分之间的边界。在上一个示例中，数组成为[4, 2, 3, 5, 6, 9]一个分区之后的对象，返回的索引为4（左指针的最后一个点）。之后，整个阵列的左侧（项目0到3）将被分区，如下图所示。

经过此遍后，数组变为[3, 2, 4, 5, 6, 9]，返回的索引为1。心律继续这样，直到对数组的所有左侧进行排序为止。然后，在数组的右侧进行相同的处理。

该quicksort()函数接受三个参数：要排序的数组，左指针应从其开始的索引和右指针应在其开始的索引。为了优化性能，如果数组具有零个或一项，则不对数组进行排序。如果数组中有两个或多个项目，则会对其进行分区。如果left小于返回的index负1，则左侧仍有要排序的项目，并quickSort()在这些项目上递归调用。同样，如果index小于right指针，则右边还有项目要排序。完成所有这些操作后，将返回数组作为结果。

function quickSort(items, left, right) {

    var index;

    if (items.length > 1) {

        left = typeof left != "number" ? 0 : left;
        right = typeof right != "number" ? items.length - 1 : right;

        index = partition(items, left, right);

        if (left < index - 1) {
            quickSort(items, left, index - 1);
        }

        if (index < right) {
            quickSort(items, index, right);
        }

    }

    return items;
}

var result = quickSort(items);
